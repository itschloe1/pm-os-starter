Getting Started
1

Claude Code 설치

Claude Code 설치 가이드를 따라 설치합니다. 터미널에서 claude 명령어가 동작하면 OK.

2

PM-OS Starter 폴더에서 실행

cd ~/PM-OS-Starter && claude click to copy
3

시작하기

Claude Code가 열리면 아래를 입력하세요. MCP 연결부터 첫 결과물까지 15분이면 됩니다.

/start click to copy
