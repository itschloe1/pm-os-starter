## 제품 구조

오픈서베이는 **Dataspace(B2B 리서치 플랫폼)** + **Ovey(B2C 리워드 앱)**이 맞물려 돌아간다.

```
Dataspace (B2B)                    Ovey (B2C)
═══════════════                    ═════════
기업 고객이 설문을                   일반 사용자가 설문에
설계/수집/분석/리포트                 참여하고 리워드를 받는
하는 리서치 플랫폼                   모바일 앱

          ┌─────────────────┐
          │  설문 응답 연결   │
  DS에서 설문 생성 ──────→ Ovey 유저에게 설문 노출
  DS에서 결과 분석 ←────── Ovey 유저가 응답 제출
          └─────────────────┘
```

Ovey 패널(20만+)이 Dataspace의 데이터 수집 인프라. 10년+ 히스토리, 누적 응답 2억건+. Synthetic Panel도 이 실제 응답 데이터 위에 구축됨. 경쟁사가 따라하기 어려운 핵심 자산.

## Dataspace

리서치 담당자, 마케터, PM을 위한 셀프서비스 리서치 플랫폼. 설문 설계부터 응답 수집, 분석, 인사이트 도출까지 하나의 플랫폼에서 수행.

### 핵심 모드

| 모드 | 설명 | 타겟 유저 |
|------|------|----------|
| **Pro Mode** | 리서처가 직접 설문 설계/분석하는 전문가 모드 | Creator (리서처) |
| **AI Mode** (4월 론칭) | AI가 리서치 전 과정 대신 실행. 첫 수익화 기점 | Creator (리서처 + non-리서처) |
| **Viewer Mode** (4월 론ㅣ) | 데이터 읽기 전용 탐색 | Viewer (경영진, 유관 부서) |

### 플랜 구조

| 플랜 | 가격 | 대상 |
|------|------|------|
| Free | 무료 | 체험 |
| Pro | ₩169K/월 | Creator (리서처), Viewer |
| Enterprise | 커스텀 | 대규모 조직 |

### 핵심 용어

| 용어 | 뜻 |
|------|-----|
| Creator seat | 리서치 설계/실행 권한 |
| Viewer seat | 데이터 열람 전용 권한 |
| AI 사고량 | AI Mode 사용 시 소비되는 크레딧 |
| Synthetic Panel | AI 기반 가상 패널, 실제 설문 전 가설 검증용 |

### 수익 모델

구독(Subscription) + 사용량(Usage-based) 혼합.

**1. 플랜 (Plan Subscription)** -- 사용 목적에 따라 기능과 가격을 구분한 구독 상품.

**2. 애드온 (Add-on)** -- 플랜 구독자가 추가 이용할 수 있는 기능. (예: API 연동)

**3. 크레딧 (Credit)** -- 사용량 기준 추가 이용.
- **응답 크레딧** (Response credit): 수집/임포트 가능 응답 건수
- **발송 크레딧** (Message credit): 이메일/SMS/알림톡 발송 건수
- **분석 크레딧** (Analysis credit): 텍스트/이미지/동영상 분석 건수

**용어 규칙:**
- "플랜을 구독한다" (O) / "플랜을 구매한다" (X)
- "크레딧을 구매한다" (O) / "크레딧을 구독한다" (X)
- "애드온을 결제한다" (O)

### 주요 기능

| 기능 | 설명 |
|------|------|
| 설문 편집 (Editor) | 다양한 문항 유형, 스크리닝 블록, 로직 설정 |
| 수집 (Collector) | 링크/이메일/SMS/알림톡 배포, 소비자 패널 연동 |
| 분석 (Analytics) | 교차분석표, 텍스트 AI, 응답 클리닝, 품질 점수 |
| 대시보드 (Dashboard) | NPS 모니터링, VoC 분석, 브랜드 인덱스 |
| 인사이트 위키 (Insight Wiki) | AI 기반 보고서 자동 생성 |
| MY패널 | 자사 패널 프로필/오디언스 관리 |
| 가이디드 솔루션 | 컨셉 테스트, 브랜드 트래킹, 광고 효과 조사, NPS 템플릿 |
| 워크플로우 (BETA) | 반복 조사 자동화 |

### 제품 구조도

```
Dataspace
├── 홈 (최근 인사이트 / 최근 대시보드 / 최근 데이터)
├── 프로젝트 (프로젝트별 데이터 관리)
├── 인사이트 위키 (AI 보고서 자동 생성)
├── 대시보드 (NPS, VoC 모니터링, 브랜드 트래킹)
├── 가이디드 솔루션 (목적별 리서치 템플릿)
├── 워크플로우 (반복 조사 자동화, BETA)
├── MY패널 (자사 패널 관리)
└── 트렌드 데이터 (카테고리별 트렌드)
```

## Ovey

설문 참여 기반 리워드 앱. 사용자가 설문에 응답하면 오베이머니를 벌고, 현금/상품으로 교환.

### 앱 구조 (5탭)

| 탭 | 기능 |
|----|------|
| 내 설문 | 참여 가능한 설문 목록 + 리워드 금액 표시 |
| 투데이 | 데일리 콘텐츠 (행운 저금통, 투표, 카드뉴스) |
| 마이오베이 | 프로필, 오베이머니 잔액, 등급, 설정 |
| 오베이샵 | 오베이머니 사용처 (현금 인출, 기프티콘, 상품 교환) |
| 더 보기 | 기타 메뉴 |

### 핵심 기능
- 설문 참여 + 오베이머니 적립 (설문당 50원~)
- 행운 저금통 (매일 5원 적립, 최대 5,000원 지급)
- 추천인 시스템 (추천 1건당 300원)
- 등급 시스템 (프로필 완성도에 따라 적립률 상승)

### 오베이머니
- 앱 내 가상 화폐. 설문 참여로 적립.
- 교환: 현금(계좌이체), 기프티콘, 상품
- 현금 인출: 최소 100원, 처리 7일 소요

### Ovey가 Dataspace에 중요한 이유

| Ovey 변경 | DS 영향 |
|-----------|---------|
| 패널 이탈 증가 | 설문 응답률 하락, 수집 품질 저하 |
| 리워드 구조 변경 | 응답 패턴 변화, 특정 세그먼트 이탈 가능 |
| 프로필 항목 변경 | 타겟팅 정확도에 영향 |
| 앱 사용성 저하 | 응답 완료율 하락 |

## 2026년 전략 테마

### #1: Monetization
- 2025년에 만든 AI 기능들을 수익화하는 해
- 4월 AI Mode 론칭이 첫 수익화 기점
- 플랜 개편: Free + Pro(₩169K) + Enterprise

### #2: 사용자 층 상향 (Creator-led Growth)
- **Creator** = 데이터 생산자. 리서처(heavy creator), Non-리서처(light creator)
- **Viewer** = 데이터 소비자. Non-리서처 및 리서처의 유관 부서, 경영진
- 핵심 가설: 리서처가 만들고, 조직 전체가 쓴다

```
Creator(리서처) → 리서치 설계/실행/분석
    → 인사이트 생산 (보고서, 대시보드, Wiki)
        → 결과를 팀/임원에게 공유 (Viewer)
            → Viewer: "우리 팀도 이렇게 하자"
                → 새 Creator seat + 새 프로젝트
```

### #3: 팔란티어 모델
- 리서치 분야의 팔란티어: 컨설팅 대체 + 소프트웨어 확장성 + AI 속도

## 타겟 고객

### ICP 요약

500인 이상 CPG/FMCG 기업에서 **NPD(신제품 개발)를 수행하는 리서처**. 눈에 보이는 제품을 만들고, 소비자 세그먼트가 다양해서 데이터 없이는 의사결정할 수 없는 환경. 에이전시 의존을 줄이고 리서치 내재화를 추진 중.

> 상세 ICP 문서: `context-library/strategy/icp-*.md`

### 산업별 우선순위

| 순위 | 산업 | 이유 |
|------|------|------|
| **1순위** | 식품/FMCG | 신제품 출시 횟수 최다, 타겟 세그먼트 가장 넓음, 다중 브랜드 구조 |
| **2순위** | 뷰티/생활용품 | CPG에 가까운 데이터 의존도, 소비자 세그먼트 다양 |
| **3순위** | 금융/보험 | 정기 VOC 조사 패턴. NPD 빈도는 CPG 대비 낮음 |
| 4순위 | IT/플랫폼 | 좋은 고객이 있지만 단일 브랜드 구조라 뾰족한 가치 제안 미정립 |

### Buyer Personas

**Persona 1: 리서치 매니저 (과장~차장)**
- FMCG/소비재 대기업에서 정기 브랜드 트래킹 운영
- "분기별 브랜드 트래킹 리포트 만드는 데만 2주 걸려요"
- Decision criteria: 데이터 품질, 자동 리포트, 패널 규모

**Persona 2: 마케팅 매니저 (대리~과장)**
- 중견 기업에서 리서치+마케팅 겸임
- "리서치팀이 따로 없어서 제가 설문 설계부터 분석까지 다 해요"
- Decision criteria: 사용 편의성, 가이디드 템플릿, 가격

**Persona 3: CX 매니저**
- 금융/서비스업에서 NPS/VOC 관리
- "NPS 점수는 나오는데 왜 그 점수인지 원인 파악이 어려워요"
- Decision criteria: NPS 드라이버 분석, 텍스트 AI 자동 분류

**Persona 4: 에이전시 담당자**
- 광고/리서치/컨설팅 에이전시에서 클라이언트 대행
- "클라이언트마다 리포트 포맷 요구가 달라서 매번 새로 만들어요"
- Decision criteria: 화이트라벨 리포트, AI 응답 품질 필터링

## Value Proposition

### 문제

리서치 담당자, 마케터, PM은 데이터 기반 의사결정을 원하지만:
- 외부 에이전시: 비용 높고(건당 ₩170만), 턴어라운드 느림(17일)
- 직접 수행: 타겟 응답자 확보 어렵고, 전문 인력 필요
- 결과: 리서치가 의사결정 속도를 못 따라감. "감"에 의존한 결정 반복.

### 솔루션

Dataspace는 설문 설계~응답 수집~분석~인사이트 도출까지 셀프서비스로 제공. Ovey 패널로 타겟 응답자를 빠르게 확보하고, AI 분석으로 시간을 단축.

### 차별화

| | 오픈서베이 | 대안 |
|---|---|---|
| vs. 리서치 에이전시 | 에이전시 17일/₩170만 → DS 4일/₩25만 | 비용, 속도, 직접 통제 |
| vs. SurveyMonkey/Typeform | 한국 시장 패널 직접 연동, 한국어 AI 분석 | 글로벌이지만 한국 패널 없음 |
| vs. Google Forms | AI 분석, 패널 연동, 교차분석, 대시보드 | 무료지만 분석 기능 없음 |
| vs. 퀄트릭스 | 고객 대응 속도, AI 기능, 가격 경쟁력 | 엔터프라이즈급이지만 고가, 대응 느림 |
